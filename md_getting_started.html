Getting Started

Connect your MYNT EYE

Congratulations on being the owner of a brand new MYNT EYE!

Unpack your MYNT EYE and plug the camera in a USB 3.0 port. The MYNT EYE is a UVC compliant camera so it should be automatically recognized by your computer.

Download and install the MYNT EYE SDK

The MYNT EYE SDK is avalible for Windows, MacOS, Linux and Nvidia Jetson boards. Get the latest MYNT EYE SDK from our Downloads page.

Windows

CUDA (Optional)

If you have Nvidia's graphic card, you could enable GPU compute capability as follows:

  • Ensure your graphic card drivers are up to date.
  • Download and install latest version of CUDA Toolkit.

MYNT EYE SDK

Run "Command Prompt (cmd)" as administrator, then:

> cd mynteye-0.1-win-x64
> install.bat

SDK samples

Prerequisites

Windows Development Kits:

  • Visual Studio
    • Visual Studio 2015
    • Note: CUDA Samples v8.0 only support vs2010-2015 now.

Add the following environment variables to PATH: 

C:\Program Files (x86)\Microsoft Visual Studio 14.0\VC\bin
C:\Program Files (x86)\MSBuild\14.0\Bin

Build Essential Tools:

  • CMake

Build

> cd samples
> mkdir build
> cd build
> cmake -DCMAKE_BUILD_TYPE=Release ^
-G "Visual Studio 14 2015 Win64" ^
..
> msbuild.exe ALL_BUILD.vcxproj /property:Configuration=Release

About SDK samples' introductions, please see "samples/README.md".

MACOS

Prerequisites

Install Homebrew to manage packages,

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Then,

*Install build tools*
$ brew install cmake git
*Install ssl for https*
$ brew install openssl

OpenCV 3.2

$ git clone https://github.com/opencv/opencv.git
$ cd opencv/
$ git checkout tags/3.2.0
$ cd opencv/
$ mkdir build
$ cd build/
$ cmake \
-DCMAKE_BUILD_TYPE=RELEASE \
-DCMAKE_INSTALL_PREFIX=/usr/local \
\
-DWITH_CUDA=OFF \
\
-DBUILD_DOCS=OFF \
-DBUILD_EXAMPLES=OFF \
-DBUILD_TESTS=OFF \
-DBUILD_PERF_TESTS=OFF \
..
$ make -j8
$ sudo make install

CUDA (Optional)

If you have Nvidia's graphic card, you could enable GPU compute capability as follows:

  • Ensure your graphic card drivers are up to date.
  • Download and install latest version of CUDA Toolkit.
$ brew tap caskroom/cask
$ brew cask install cuda
$ brew cask info cuda

MYNT EYE SDK

$ cd mynteye-0.1-mac-x64/
$ ./install.sh
$ source ~/.bash_profile

If your OpenCV or CUDA is installed in a different path, could specify cuda & opencv library search paths in ~/.bash_profile like this: 

RPATH_CUDA=/usr/local/cuda/lib
RPATH_OPENCV=/usr/local/lib
export DYLD_LIBRARY_PATH=$RPATH_CUDA:$DYLD_LIBRARY_PATH
export DYLD_LIBRARY_PATH=$RPATH_OPENCV:$DYLD_LIBRARY_PATH

Build SDK samples

$ cd samples/
$ mkdir build
$ cd build/
$ cmake -DCMAKE_BUILD_TYPE=Release ..
$ make

About SDK samples' introductions, please see "samples/README.md".

LINUX

Prerequisites

# Install build tools
$ sudo apt-get install build-essential cmake git
# Install OpenCV dependencies
$ sudo apt-get install pkg-config libgtk2.0-dev
# Install udev for usb, ssl for https, v4l for video
$ sudo apt-get install libudev-dev libssl-dev libv4l-dev v4l-utils

OpenCV 3.2

$ git clone https://github.com/opencv/opencv.git
$ cd opencv/
$ git checkout tags/3.2.0
$ cd opencv/
$ mkdir build
$ cd build/
$ cmake \
-DCMAKE_BUILD_TYPE=RELEASE \
-DCMAKE_INSTALL_PREFIX=/usr/local \
\
-DWITH_CUDA=OFF \
\
-DBUILD_DOCS=OFF \
-DBUILD_EXAMPLES=OFF \
-DBUILD_TESTS=OFF \
-DBUILD_PERF_TESTS=OFF \
..
$ make -j$(nproc --all)
$ sudo make install

CUDA (Optional)

If you have Nvidia's graphic card, you could enable GPU compute capability as follows:

  • Ensure your graphic card drivers are up to date.
  • Download and install latest version of CUDA Toolkit.

MYNT EYE SDK

$ cd mynteye-0.1-linux-x64/
$ ./install.sh
$ source ~/.bashrc

If your OpenCV or CUDA is installed in a different path, before running ./install.sh, you could change the following variables in ./install.sh: 

RPATH_CUDA=/usr/local/cuda/lib64
RPATH_OPENCV=/usr/local/lib

Build SDK samples

$ cd samples/
$ mkdir build
$ cd build/
$ cmake -DCMAKE_BUILD_TYPE=Release ..
$ make

About SDK samples' introductions, please see "samples/README.md".
